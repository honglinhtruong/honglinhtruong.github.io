My consulting services

I offer consulting services for big data analytics, applied machine learning, cloud computing, edge computing and IoT, as our core engineering knowledge. We consult how to use these enabling technologies for digital transformation and future of work for enterprises.

I can help customers to build their software services and infrastructures. Especially, my core knowhow is on software/microservice engineering and on intelligent analytics for large-scale systems, IoT, big data and clouds. At the heart of my consulting work are also intensive research, training, and novel solutions for large-scale service and machine learning engineering (performance, reliability, scalability) and DevOps methods for productivity. The technologies I am working on are:

  • Big data systems: Datalake technologies (Apache Hudi, Delta lake), Hadoop ecosystems, Apache Spark, ELK stack (ElasticSearch, Logstash and Kibana), TICK stack (Telegraf, InfluxDB, Chronograf, and Kapacitor), Apache Kafka, Mongodb, Cassandra, RabbitMQ, Apache NiFi, and Redis
  • IoT and edge computing: MQTT, Beacon/BLE, RFID, LoraWAN, Waspmote, Raspberry Pi
  • Large-scale cloud and microservices systems engineering: OpenStack, Docker, Kubernetes, Amazon/Azure/Google infrastructures, and serverless computing
  • Analytics: Data science and machine learning with batch/stream analytics and data mining/machine learning algorithms: Apache Flink/Spark, scikit-learn, Dask, TensorFlow/Keras
  • GitHub repositories
