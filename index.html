Our Services

We offer consulting services for big data analytics, applied machine learning, cloud computing, edge computing and IoT, as our core engineering knowledge. We advise how to use these enabling technologies for digital transformation and future of work for enterprises.

We help customers to build their software services and infrastructures. Especially, our core knowhow is on software/microservice engineering and on intelligent analytics for large-scale systems, IoT, big data and clouds. At the heart of our offerings are also intensive research, training, and novel solutions for large-scale service and machine learning engineering (performance, reliability, scalability) and DevOps methods for productivity. The technologies we are working on are:

  • Big data systems: Hadoop ecosystems, Apache Spark, ELK stack (ElasticSearch, Logstash and Kibana), TICK stack (Telegraf, InfluxDB, Chronograf, and Kapacitor), Apache Kafka, Apache Apex, Mongodb, Cassandra, RabbitMQ, Apache NiFi, and Redis
  • IoT and edge computing: MQTT, Beacon/BLE, RFID, LoraWAN, Waspmote, Raspberry Pi
  • Large-scale cloud and microservices systems engineering: OpenStack, Docker, Kubernetes, Amazon/Azure/Google infrastructures, and serverless computing
  • Analytics: Data science and machine learning with batch/stream analytics and data mining/machine learning algorithms: Apache Flink/Spark, scikit-learn, Dask, TensorFlow/Keras
  • Blockchain: blockchain technologies and their software and services engineering (e.g, with Ethereum, Hyperledger, BigChainDB)
  • GitHub repositories
